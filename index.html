Ajustes del Sistema

Modo Sin Conexión

Si activas esta opción, el sistema utilizará tu última memoria guardada para permitirte registrar ventas cuando se caiga el internet. Las ventas quedarán en cola y se subirán cuando regreses.

Base de Datos

La inicialización creará las hojas de cálculo necesarias en tu Google Sheet si no existen. Útil para la primera configuración.

Copias de Seguridad (Exportar / Importar)

Descarga un respaldo completo de productos, categorías y ventas en un solo archivo JSON, o restaura uno anterior. Es el mismo formato que se envía automáticamente por correo a los Responsables.

⚠️ Esto reemplaza los productos, categorías y ventas actuales por los del archivo. No se puede deshacer.

Envía por correo, en este momento, la misma copia de seguridad a los responsables que tengan activado "Copia de seguridad JSON/CSV" en la sección Responsables.

Anular Venta

Probar Envío de Correo

Se enviará un correo de prueba a con el reporte que elijas, sin esperar la fecha programada. Si eliges "Probar TODOS", se enviará un correo de prueba por cada tipo de reporte (uno por cada uno).

Ventas Pendientes de Sincronizar

Estas ventas se guardaron en este dispositivo mientras no había conexión. Si una muestra un conflicto de stock (no se pudo sincronizar porque ya no hay suficiente inventario real), puedes descartarla manualmente para dejar de intentar subirla.
